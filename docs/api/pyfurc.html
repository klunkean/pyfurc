class pyfurc.core.PhysicalQuantity(name, quantity_type='parameter', value=0.0)¶

Bases: sympy.core.symbol.Symbol

Fundamental class for degrees of freedom and loads.

Parameters

  • name (str) – The name that will be displayed in outputs. When working in a jupyter notebook, LaTeX symbols can be used, e.g. \varphi.

  • quantity_type (str, optional) – One of load, dof, or parameter, by default parameter

  • value (float, optional) – Initial value if quantity_type is load or dof. Value if quantity_type is parameter. Default is 0.0.
