Developer Guide¶

If you want to contribute to pyfurc, fork the git repository and start from there.

Tools¶

Poetry¶

Dependency management, packaging as well as publishing on PyPi is done using Poetry. Poetry also provides easy to use virtual environments for developing.

Sphinx¶

The documentation is written in reStructuredText and built using Sphinx. The docs from the master branch are hosted on readthedocs.

pytest¶

Pytest is used for unit testing.
